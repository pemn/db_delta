📌 Description

create a custom waterfall chart from block model data

📸 Screenshot

📝 Parameters

name optional description
input_a,input_b ❎ path to data in a supported format
keys_a,keys_b ❎ primary key fields for indexing
groups_a,groups_b ❎ group fields for comparison
value_a,value_b ❎ numeric field for totals
condition ☑️ only rows that evaluate to true will be used
mode waterfall generate a waterfall+candlestick chart
compare generate a custom chart optimized for comparison between classes
output_table ☑️ path to save the comparison table as xlsx
output_chart ☑️ path to save the buildup chart as png/pdf
display show the result in a popup window

📓 Notes

  • The number of keys must be the same both for A and B
  • The number of groups must be the same for both A and B
  • condition will apply to both A and B
  • in the output, d_0 is the internal variation of B - A and d_1 is A - B. Both only when > 0.
A B d_0 d_1
9000 8000 0 1000
8000 9000 1000 0
